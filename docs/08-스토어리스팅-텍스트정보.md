## 3. 전체 설명 (Full Description)

**최대 4000자**

### 추천 설명문

```
🏓 PingTalk - 워치 연동 스마트 점수판

탁구, 배드민턴 등 1:1 게임을 위한 점수판 앱입니다. 스마트워치와 연동하여 게임에 집중하면서도 점수를 편리하게 관리할 수 있습니다.

✨ 주요 기능

📱 직관적인 점수판
• HOME/AWAY 양쪽 점수를 한눈에 확인
• 큰 숫자로 명확하게 표시되는 점수판
• 가로/세로 모드 모두 지원

⌚ 워치 연동
• 스마트워치에서 간편하게 점수 조작
• +1/-1 버튼으로 빠른 점수 변경
• 실시간 동기화로 폰과 워치가 항상 일치
• 연결 상태를 명확하게 표시

🔄 실수 방지 기능
• Undo 기능으로 실수한 점수 되돌리기
• 점수 히스토리를 통한 안전한 관리
• 리셋 시 확인 다이얼로그로 오동작 방지

💾 자동 저장
• 앱을 종료해도 점수가 자동으로 저장
• 최근 경기 기록 보관
• 앱 재실행 시 이전 점수 유지

⚙️ 커스터마이징
• 플레이어 이름 변경 (HOME/AWAY)
• 점수 단위 설정
• 게임 규칙 설정

🎮 지원 게임
• 탁구
• 배드민턴
• 기타 1:1 점수제 게임

📱 워치 지원
• Wear OS (Galaxy Watch 등)
• 워치에서 점수 조작 및 확인
• 폰 없이도 워치에서 점수 관리 가능

🔒 개인정보 보호
• 모든 데이터는 로컬에만 저장
• 인터넷 연결 불필요
• 계정 가입 불필요

사용 방법:
1. 앱을 실행하고 플레이어 이름을 설정하세요
2. 스마트워치와 연결하세요 (자동 연결)
3. 게임을 시작하고 워치에서 점수를 조작하세요
4. 실수가 있다면 Undo 버튼으로 되돌리세요

PingTalk로 게임에 집중하고, 점수 관리는 우리에게 맡기세요!
```

**문자 수**: 약 1,200자 ✅

   - 스마트워치를 착용하고 앱을 실행하면 자동으로 연결됩니다

4. 경기 종료
   - 리셋 버튼으로 새로운 경기를 시작합니다
   - 확인 다이얼로그로 실수 방지

• 폰은 테이블에 두고 큰 화면으로 점수를 확인하세요
• 실수가 있다면 바로 Undo 버튼을 누르세요
• 경기 기록은 자동으로 저장되므로 나중에 확인할 수 있습니다
