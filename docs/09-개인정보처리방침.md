# 개인정보처리방침 (Privacy Policy)

이 문서는 PingBoard 앱의 개인정보처리방침입니다. 구글 플레이 스토어 배포 시 이 내용을 웹페이지로 호스팅하여 URL을 제공해야 합니다.

---

## 개인정보처리방침

**최종 업데이트: 2025년 1월 31일**

PingBoard("회사", "우리", "저희")은(는) 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 PingBoard 앱("앱", "서비스")을 사용할 때 수집, 사용, 공유되는 정보에 대해 설명합니다.

### 1. 수집하는 정보

**PingBoard은 사용자의 개인정보를 수집하지 않습니다.**

앱은 다음과 같은 정보를 수집하지 않습니다:
- 이름, 이메일 주소, 전화번호 등 개인 식별 정보
- 위치 정보
- 기기 정보
- 사용 패턴 또는 분석 데이터
- 기타 개인정보

### 2. 로컬 저장 데이터

앱은 게임 점수 데이터를 사용자의 기기에만 로컬로 저장합니다:
- 현재 경기 점수
- 최근 경기 기록
- 앱 설정 (플레이어 이름, 점수 단위 등)

**이러한 데이터는:**
- 사용자의 기기에만 저장됩니다
- 인터넷을 통해 전송되지 않습니다
- 서버에 업로드되지 않습니다
- 다른 사용자와 공유되지 않습니다
- 회사나 제3자에게 전송되지 않습니다

### 3. 데이터 사용 목적

로컬에 저장된 데이터는 오직 다음 목적으로만 사용됩니다:
- 앱 기능 제공 (점수 관리, 경기 기록)
- 앱 재실행 시 이전 상태 복원
- 사용자 설정 유지

### 4. 데이터 보관 및 삭제

- 모든 데이터는 사용자의 기기에만 저장됩니다
- 사용자가 앱을 삭제하면 모든 데이터가 함께 삭제됩니다
- 사용자는 언제든지 앱 내에서 데이터를 초기화할 수 있습니다

### 5. 제3자 서비스

PingBoard은 제3자 분석 도구, 광고 네트워크, 또는 기타 데이터 수집 서비스를 사용하지 않습니다.

### 6. 권리 및 선택

사용자는 다음 권리를 가집니다:
- 앱 삭제: 언제든지 앱을 삭제하여 모든 데이터를 제거할 수 있습니다
- 데이터 초기화: 앱 내에서 리셋 기능을 통해 모든 데이터를 초기화할 수 있습니다

### 7. 아동의 개인정보 보호

PingBoard은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 앱은 모든 연령대의 사용자가 안전하게 사용할 수 있습니다.

### 8. 개인정보처리방침 변경

개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있는 경우 앱 내에서 또는 웹사이트를 통해 알려드립니다.

### 9. 연락처 정보

개인정보처리방침에 대한 질문이나 우려사항이 있으시면 다음으로 연락해 주세요:

## 웹 호스팅 가이드

구글 플레이 스토어에 개인정보처리방침 URL을 제공하기 위해 다음 방법 중 하나를 사용할 수 있습니다.

### 방법 1: GitHub Pages (권장)

#### 1단계: GitHub 저장소에 파일 추가

1. 프로젝트 루트에 `docs/privacy-policy.html` 파일 생성
2. 아래 HTML 템플릿 사용하여 개인정보처리방침 페이지 생성

#### 2단계: GitHub Pages 활성화

1. GitHub 저장소 → Settings → Pages
2. Source를 "Deploy from a branch" 선택
3. Branch를 "main" (또는 "master"), 폴더를 "/docs" 선택
4. Save 클릭

#### 3단계: URL 확인

- URL 형식: `https://[사용자명].github.io/[저장소명]/privacy-policy.html`
- 예시: `https://01000-you.github.io/pingtalk/privacy-policy.html`

### 방법 2: 개인 웹사이트

기존 웹사이트가 있다면:
1. 개인정보처리방침 페이지 생성
2. 공개적으로 접근 가능한 URL 제공
3. 예시: `https://yourwebsite.com/privacy-policy`

### 방법 3: 무료 호스팅 서비스

다음 무료 호스팅 서비스를 사용할 수 있습니다:
- **Netlify**: https://www.netlify.com
- **Vercel**: https://vercel.com
- **GitHub Pages**: https://pages.github.com
