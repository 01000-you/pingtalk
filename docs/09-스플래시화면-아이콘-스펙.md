# Android 스플래시 화면 아이콘 스펙

## Android 12+ (API 31+) 스플래시 화면 아이콘 스펙

### 기본 스펙
- **권장 크기**: 288dp x 288dp (중앙 아이콘 영역)
- **실제 픽셀 크기** (DPI별):
  - mdpi: 288x288px
  - hdpi: 432x432px
  - xhdpi: 576x576px
  - xxhdpi: 864x864px
  - xxxhdpi: 1152x1152px

### 특징
- 아이콘은 **원형 마스크**가 자동 적용됨
- 배경색과 대비가 있어야 함 (현재 배경: #0B1220)
- 투명 배경 가능
- 아이콘은 중앙에 표시되며, 시스템이 자동으로 크기 조절

### 현재 설정
- `windowSplashScreenAnimatedIcon`: `@mipmap/ic_launcher` 사용
- `windowSplashScreenIconBackgroundColor`: `#0B1220` (어두운 배경)
- 현재 ic_launcher는 192x192px (xxxhdpi 기준)이므로 시스템이 자동으로 크기 조절

### 작은 아이콘을 원하는 경우

#### 방법 1: 별도의 작은 아이콘 리소스 생성
1. **288dp 크기의 아이콘 이미지 준비**
   - xxxhdpi 기준: 1152x1152px
   - 투명 배경 또는 어두운 배경과 대비되는 색상

2. **drawable 리소스로 추가**
   ```
   res/drawable/splash_icon_small.xml
   ```
   또는
   ```
   res/drawable-xxxhdpi/splash_icon_small.png (1152x1152px)
   ```

3. **styles.xml에서 사용**
   ```xml
   <item name="android:windowSplashScreenAnimatedIcon">@drawable/splash_icon_small</item>
   ```

#### 방법 2: launch_background.xml에 작은 아이콘 추가 (Android 11 이하용)
- `launch_background.xml`에 72dp 크기의 아이콘 추가 (이미 적용됨)
- Android 12+에서는 `windowSplashScreenAnimatedIcon`이 우선 적용됨

### 권장 사항
- **최적 크기**: 288dp x 288dp (1152x1152px @ xxxhdpi)
- **형식**: PNG (투명 배경 가능) 또는 Vector Drawable
- **색상**: 어두운 배경(#0B1220)과 대비되는 밝은 색상 권장
- **디자인**: 원형 마스크가 적용되므로 중앙에 중요한 요소 배치
