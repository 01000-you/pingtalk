# 릴리즈 노트 간결 버전 (한 번에 붙여넣기용)

구글 플레이 콘솔에 바로 복사해서 붙여넣을 수 있는 간결 버전입니다.

---

## 전체 언어 버전 (한 번에 붙여넣기)

아래 내용을 전체 복사하여 구글 플레이 콘솔의 출시 노트 필드에 붙여넣으세요:

```

1:1 게임을 위한 스마트 점수판 앱입니다.

✨ 주요 기능
• 직관적인 점수판 UI
• 스마트워치(Wear OS) 연동
• Undo 및 Reset 기능
• 세트 점수 히스토리 관리
• 자동 저장

워치에서 완전한 점수 컨트롤 제공!

이제 게임에 집중하고, 점수 관리는 PingBoard에게 맡기세요!
</ko-KR>
```

## 사용 방법

1. 위의 "전체 언어 버전" 섹션에서 전체 텍스트를 복사
2. 구글 플레이 콘솔의 출시 노트 필드에 붙여넣기
3. 저장

**참고**: 구분선이나 추가 설명 없이 순수하게 언어 태그와 내용만 포함되어 있어 바로 사용할 수 있습니다.
