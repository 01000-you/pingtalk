1:1 게임을 위한 스마트 점수판 앱이 드디어 출시되었습니다!

✨ 주요 기능
• 직관적인 점수판 UI로 HOME/AWAY 점수를 한눈에 확인

• Undo와 Reset 버튼으로 완전한 점수 컨트롤 제공
• 실시간 동기화로 폰과 워치가 항상 일치

🎮 지원 게임
• 탁구
• 배드민턴
• 기타 1:1 점수제 게임

## 구글 플레이 콘솔 입력 방법

1. **출시 노트 필드**에 위의 언어별 버전을 각각 입력
2. 각 언어 태그(`<ko-KR>`, `<en-US>`, `<ja-JP>`, `<zh-CN>`)를 정확히 입력
3. 태그 안에만 텍스트가 있도록 주의
4. 저장

### 예시: 여러 언어를 한 번에 입력

```
<ko-KR>
한국어 출시 노트 내용
</ko-KR>
<en-US>
English release notes content
</en-US>
<ja-JP>
日本語リリースノート内容
</ja-JP>
<zh-CN>
中文发布说明内容
</zh-CN>
```

## 참고사항

- 각 언어 태그는 정확히 입력해야 합니다: `<ko-KR>`, `<en-US>`, `<ja-JP>`, `<zh-CN>`
- 태그는 대소문자를 구분합니다
- 모든 텍스트는 해당 언어 태그 안에만 있어야 합니다
- 태그 밖에 공백이나 텍스트가 있으면 오류가 발생합니다
- 이모지는 사용 가능하지만, 일부 특수 문자는 문제를 일으킬 수 있습니다
- 출시 노트는 사용자에게 표시되므로 친근하고 이해하기 쉽게 작성하세요
